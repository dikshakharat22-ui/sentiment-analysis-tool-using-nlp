üìä Project Overview

This project implements two main tasks:

Binary & Multi-class Sentiment Analysis

Analyzes text to determine sentiment (Positive, Negative, or Neutral) using advanced NLP techniques.

Movie Recommendation System

Provides movie recommendations using clustering algorithms based on genres and ratings.

üìö Datasets Used

  • IMDB Reviews Dataset: Binary sentiment labels (Positive, Negative reviews)
  • Tweets Dataset: 3-way sentiment labels (Pleasant, Unpleasant, Neutral)

ü§ñ Machine Learning Approaches

Traditional ML Models

  • Random Forest: Collection of decision trees for feature importance analysis
  • Logistic Regression: Linear classification with feature weights for sentiment discrimination

Deep Learning NLP Models

  1. Recurrent Neural Network (RNN) - The Baseline: Uses RNN nodes with learnable parameters and 100-D word embeddings
  2. Average Sentiment Network (AvgNet): Uses Global Average Pooling instead of RNN for faster processing
  3. Convolutional Neural Network (CNN) - CNet: CNN-based approach for effective sentiment classification
Best Performance: AvgNet and CNet achieved excellent accuracy with effective training convergence. AvgNet scored 92.51% accuracy on the Tweets dataset.

üé¨ Recommendation System

The movie recommendation system uses data from the OMDB API with features including genres and ratings.

Clustering Models Used:

  • K-means Clustering
  • Agglomerative Clustering
  • DBSCAN Clustering
